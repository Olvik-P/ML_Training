## Введение

Зачастую данные приходят с названиями столбцов, индексов или другими соглашениями об именовании, которые нас не устраивают. В этом случае вы узнаете, как использовать функции pandas для изменения названий неугодных элементов на более подходящие.

Вы также изучите, как объединять данные из нескольких DataFrame и/или Series.

## Переименование

Первая функция, которую мы здесь рассмотрим — это `rename()`, позволяющая изменять имена индексов и/или названия столбцов. Например, чтобы изменить столбец `points` в нашем наборе данных на `score`, мы сделаем следующее:

```python
reviews.rename(columns={'points': 'score'})
```

`rename()` позволяет переименовывать значения индекса или столбцов, указывая соответственно параметры `index` или `column`. Она поддерживает различные форматы ввода, но обычно удобнее всего использовать словарь Python. Вот пример переименования некоторых элементов индекса:

```python
reviews.rename(index={0: 'firstEntry', 1: 'secondEntry'})
```

## Объединение

При выполнении операций с набором данных иногда возникает необходимость объединять различные DataFrame и/или Series нетривиальными способами. В Pandas для этого есть три основных метода. В порядке возрастания сложности это: `concat()`, `join()` и `merge()`. Большую часть того, что может делать `merge()`, можно проще выполнить с помощью `join()`, поэтому мы опустим `merge()` и сосредоточимся на первых двух функциях.

Самый простой метод объединения — `concat()`. Получив список элементов, эта функция «склеивает» их вдоль оси.

Это полезно, когда у нас есть данные в разных объектах DataFrame или Series, но с одинаковыми полями (столбцами). Один из примеров — набор данных YouTube Videos, который разделяет данные по стране происхождения (например, Канада и Великобритания). Если мы хотим изучать несколько стран одновременно, мы можем использовать `concat()`, чтобы объединить их:

```python
canadian_youtube = pd.read_csv("../input/youtube-new/CAvideos.csv")
british_youtube = pd.read_csv("../input/youtube-new/GBvideos.csv")

pd.concat([canadian_youtube, british_youtube])
```

Средний по сложности метод объединения — `join()`. `join()` позволяет объединять разные объекты DataFrame, у которых есть общий индекс. Например, чтобы получить видео, которые оказались в трендах в один и тот же день как в Канаде, так и в Великобритании, мы можем сделать следующее:

```python
left = canadian_youtube.set_index(['title', 'trending_date'])
right = british_youtube.set_index(['title', 'trending_date'])

left.join(right, lsuffix='_CAN', rsuffix='_UK')
```
