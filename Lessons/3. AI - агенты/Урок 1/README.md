# 🤖 Урок 1: Создаём своего первого AI-агента

## Теория: что такое AI-агент?

**Агент — это как продавец в киоске со шпаргалкой.**

Представь: ты подходишь к киоску и задаёшь вопрос. Продавец:
1. **Слышит тебя** (получает ввод)
2. **Смотрит в шпаргалку** (ищет ответ в памяти)
3. **Отвечает** (выводит результат)
4. **Ждёт следующий вопрос** (цикл)

В программировании это называется **«цикл + инструкция + память»**:

| Компонент | В коде | Аналогия |
|-----------|--------|----------|
| **Цикл** | `while True` | Продавец не уходит домой, пока ты не скажешь «пока» |
| **Инструкция** | Функция `find_answer()` | Правило: «ищи ключевое слово и отвечай» |
| **Память** | Словарь `facts` | Шпаргалка с фактами |

Вот и всё! Никакой магии. Настоящие AI-агенты (ChatGPT, Алиса) работают по той же схеме, только вместо словаря у них нейросеть, а вместо `find_answer()` — сложная математика. Но основа та же.

## Код: simple_agent.py

Файл уже создан: [`simple_agent.py`](simple_agent.py)

Открой его в VS Code или любом текстовом редакторе. Код полностью на русском с комментариями — читай их по порядку.

**Что делает код:**
1. Хранит словарь фактов (тема → ответ)
2. Ищет ключевые слова в вопросе пользователя
3. Отвечает, если слово найдено
4. Говорит «не знаю», если слова нет в словаре
5. Работает в бесконечном цикле, пока не скажешь «выход»

## 🎯 Упражнение для новичка

Сделай агента **умнее**! Открой файл [`simple_agent.py`](simple_agent.py) и:

### Задание 1. Добавь 2 новых факта

Найди в коде словарь `facts` (строки 20–30) и добавь туда две новые записи. Например:

```python
"еда":        "Я люблю электричество. Это моя еда!",
"кино":       "Я не смотрю кино, но слышал, что 'Матрица' — хороший фильм.",
```

### Задание 2. Добавь новое правило ответа

Сейчас агент ищет **одно** ключевое слово. А если вопрос содержит сразу два слова?  
Добавь в функцию `find_answer()` проверку: если в вопросе есть слово **«как»** — отвечай специальной фразой.

Подсказка: добавь в начало функции `find_answer()` такой код:

```python
if "как" in question_lower:
    return "Ого, вопрос 'как'! Я пока умею отвечать только на 'что' и 'кто'."
```

### Проверь себя

После изменений запусти агента снова и спроси:

```
Ты: что ты ешь?
Ты: какой фильм посоветуешь?
Ты: как дела?
```

Если агент отвечает на все три вопроса — ты справился! 🎉
