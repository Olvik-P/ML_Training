# Использование Pandas для знакомства с данными

Первым шагом в любом проекте машинного обучения является знакомство с данными. Для этого вы будете использовать библиотеку Pandas. Pandas — это основной инструмент, который специалисты по данным используют для исследования и обработки данных. Большинство людей сокращают pandas в своём коде как `pd`. Мы делаем это с помощью команды

```bash
import pandas as pd
```

Самая важная часть библиотеки Pandas — это DataFrame. DataFrame содержит данные, которые можно представить в виде таблицы. Это похоже на лист в Excel или таблицу в базе данных SQL.

Pandas предоставляет мощные методы для большинства операций, которые вы захотите выполнить с данными такого типа.

В качестве примера мы рассмотрим данные о ценах на жильё в Мельбурне, Австралия. В практических упражнениях вы примените те же процессы к новому набору данных, содержащему цены на жильё в Айове.

Пример данных (Мельбурн) находится по пути `../input/melbourne-housing-snapshot/melb_data.csv`.

Мы загружаем и исследуем данные с помощью следующих команд:

```bash
# сохраняем путь к файлу в переменной для удобства доступа
melbourne_file_path = '../input/melbourne-housing-snapshot/melb_data.csv'
# читаем данные и сохраняем в DataFrame с именем melbourne_data
melbourne_data = pd.read_csv(melbourne_file_path) 
# выводим сводную информацию о данных Мельбурна
melbourne_data.describe()
```

## Интерпретация описания данных

Результаты показывают 8 чисел для каждого столбца в исходном наборе данных. Первое число, **count**, показывает, сколько строк имеют непустые значения.

Пропущенные значения возникают по многим причинам. Например, размер второй спальни не будет собираться при обследовании дома с одной спальней. Мы вернёмся к теме пропущенных данных позже.

Второе значение — **mean** (среднее), то есть среднее арифметическое. Под ним **std** — стандартное отклонение, которое измеряет, насколько численно разбросаны значения.

Чтобы интерпретировать значения **min**, **25%**, **50%**, **75%** и **max**, представьте, что каждый столбец отсортирован от наименьшего к наибольшему значению. Первое (наименьшее) значение — это **min**. Если пройти четверть пути по списку, вы найдёте число, которое больше 25% значений и меньше 75% значений. Это значение **25%** (произносится «25-й процентиль»). 50-й и 75-й процентили определяются аналогично, а **max** — это наибольшее число.
